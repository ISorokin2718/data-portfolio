E-commerce Data Analysis

Анализ данных интернет-магазина для выявления закономерностей продаж и поведения клиентов.

Были использованы следующие технологии:
1) Python
2) pandas
3) matplotlib
4) numpy

Что сделано:

1) Очистка данных
2) Анализ продаж
3) Визуализация
4) Расчёт ключевых метрик

Визуализация:
ecommerce-analysis/images/revenue.png
ecommerce-analysis/images/pareto_products.png

Основные выводы:

1) Продажи имеют сезонный характер с пиком осенью
2) Основная выручка приходится на небольшое число клиентов
3) Некоторые товары формируют значительную долю дохода

Как запустить:

```bash
pip install -r requirements.txt
jupyter notebook
```
